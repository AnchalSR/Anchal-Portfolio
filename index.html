WhatsApp Chat Analyzer

Chat Insight Dashboard

Developed a Python-based WhatsApp Chat Analyzer to extract insights from raw chat exports, using regular expressions and pandas for efficient parsing and structuring of over 740 messages. Integrated NLP techniques for sentiment analysis and user activity tracking, and visualized trends with matplotlib, seaborn, and wordcloud. The solution features an interactive time-series dashboard and is deployed on Streamlit for accessible, real-time analysis.

Developed FavMovie, a content-based movie recommendation system in Python that analyzes TMDB movie and credits data to suggest similar films based on genres, keywords, cast, and crew. The project features robust data wrangling, NLP-driven tag engineering, and cosine similarity for recommendations. the entire solution was deployed as an interactive web app using Streamlit, enabling users to get instant movie suggestions through a simple, user-friendly interface
